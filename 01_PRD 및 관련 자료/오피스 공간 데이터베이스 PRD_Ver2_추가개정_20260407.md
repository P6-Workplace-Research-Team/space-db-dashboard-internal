## 1. 개정 목적

본 개정안은 기존 `오피스 공간 데이터베이스 PRD Ver 2`에 반영된 내용 외에, 실제 HTML 대시보드 구현 및 사용자 피드백 과정에서 추가로 확정된 사항을 문서화하기 위한 부속 개정안이다.

이번 추가 개정의 핵심 범위는 다음과 같다.

- 필터 구조 고도화 및 다중 선택 지원
- 좌측 필터, 우측 바로가기, 헤더 탭 중심의 탐색 UX 정비
- 개괄 탭의 정보 표현 방식 개편
- 업무공간 : 일반 탭의 KPI 및 차트 구조 전면 개편
- 등간격 지표 구간 기준 확정

## 2. PRD 반영 필요 항목

### 2.1 3.1 프로젝트 조건 입력 및 맞춤 필터 개정

기존 PRD의 다음 문장을 수정한다.

기존:

> 이때 사용하는 변수는 인원구간, 가용면적구간, 연도, 주소, 권역, 업종, 대기업여부, 소속대기업, 사업장 유형이다.

개정:

> 이때 사용하는 변수는 연도, 가용면적구간, 인원구간, 프라임오피스, 참고도면, 주소, 권역, 산업군, 사업장 유형, 대기업 여부, 소속 대기업이다.
>
> 필터는 다음 3개 카테고리로 구분하여 제공한다.
>
> - 도면 관련 필터링: 연도, 가용면적구간, 인원구간, 프라임오피스, 참고도면
> - 위치 관련 필터: 주소, 권역
> - 기업 관련 필터: 산업군, 사업장 유형, 대기업 여부, 소속 대기업
>
> 동일 필드 내에서는 다중 선택이 가능하며, 같은 필드 내 다중 선택은 OR 조건, 서로 다른 필드 조합은 AND 조건으로 동작한다.

추가 반영 기준:

- 화면상 `업종` 라벨은 `산업군`으로 표기한다.
- `프라임오피스`는 `O`, `X`, `사옥`을 각각 별도 옵션으로 제공한다.

### 2.2 3.2 데이터셋을 기준으로 계산식 산출 개정

다음 원칙을 추가한다.

> 비율형 지표 중 연도별 추이를 보여주는 지표는 `연도 내 전체 합산 후 비율`이 아니라, `도면별 비율을 먼저 산출한 뒤 그 평균`을 사용한다.

적용 대상:

- 개괄: 연도별 오피스 공간 프로그램 구성 비율
- 업무공간 : 일반: 데스크 형태별 사용 비중, 데스크 너비별 사용 비중
- 소셜공간: 연도별 소셜 공간 내 유형별 구성 비율

### 2.3 회의공간 지표 정의 보완

다음 내용을 PRD에 명시적으로 추가한다.

> 회의공간 관련 지표는 모두 `기타회의공간`을 제외한다.

그리고 다음 기준을 명확히 한다.

- 회의실 평균 개소: 회의실이 1개 이상 존재하는 도면만 평균 분모에 포함
- 회의실 규모별 평균 개소: 특정 규모 회의실이 있는 도면만이 아니라, 회의실이 1개 이상 존재하는 도면 전체를 공통 분모로 사용

### 2.4 소셜공간 지표 정의 보완

기존 PRD의 `소셜 공간 내 유형별 구성 비율` 정의를 다음처럼 보완한다.

개정:

> 프로그램이 공용인 네모를 소셜공간 계산의 기준으로 사용한다.  
> 소셜 공간 내 유형별 구성 비율은 각 도면에서 `세부유형 면적 / 공용 총면적`을 먼저 계산한 뒤, 이를 연도별로 평균하여 산출한다.

### 2.5 지원공간 평균 개소 기준 보완

다음 문장을 추가한다.

> 지원공간 평균 개소 지표는 해당 공간이 존재하는 도면만 평균에 포함한다.

적용 대상:

- 선택업무공간_실 평균 개소
- 폰부스 및 선택업무공간_실 평균 개소
- OA 평균 개소

## 3. 4. UX / 화면 구성 요구사항 추가 개정

### 3.1 전체 화면 구조 개정

기존의 단순 상단 필터 구조 설명에 더해 다음을 반영한다.

- 헤더: 서비스 타이틀과 탭 네비게이션을 함께 제공
- 좌측: 필터 전용 고정 사이드 패널
- 중앙: 탭별 지표 콘텐츠
- 우측: 현재 탭 내 주요 지표로 이동할 수 있는 바로가기 패널

추가 요구사항:

- 필터 패널은 접기/펼치기가 가능해야 한다.
- 우측 바로가기는 현재 보고 있는 카드 위치와 연동되어 하이라이트되어야 한다.
- 바로가기 클릭 시 카드 제목이 헤더에 가려지지 않도록 스크롤 오프셋을 적용해야 한다.

### 3.2 개괄 탭 시각화 방식 개정

기존 `그래프 종류를 선택한다` 수준의 서술을 다음처럼 구체화한다.

- 인원 구간별 도면 수: 연도 x 인원구간 히트맵 표
- 가용 면적 구간별 도면 수: 연도 x 가용면적구간 히트맵 표
- 인원 구간별 가용 면적 범위: 중앙값, Q1, Q3 표 + 박스플롯
- 가용 면적 구간별 인원 범위: 중앙값, Q1, Q3 표 + 박스플롯
- 오피스 공간 프로그램 구성 비율 추이: 전체 1개 + 2025→2020 연도별 누적 가로 막대

표기 원칙:

- 주요 값은 툴팁 없이 바로 읽히도록 시각 요소 안에 직접 표기한다.
- 건수 표시는 자연수로 표기한다.

### 3.3 업무공간 : 일반 탭 구조 개정

업무공간 : 일반 탭의 상단 KPI와 주요 카드 구성을 다음과 같이 명시한다.

#### 상단 KPI

- 업무 공간 비중
- 평균 인당 면적
- 주요 등간격(일자형)
- 주요 등간격(L자형)
- 주요 데스크 형태
- 주요 데스크 너비

#### 주요 카드 구성

- 인당 면적
- 자율좌석제 시행 비중
- 데스크 형태별 사용 비중
- 데스크 너비별 사용 비중
- 팀장석 구성 유형
- 등간격 선택 순위
- 그룹업무공간 보유 비중

그리고 기존에 있던 `주요 데스크 채택 현황` 카드는 제거한다.

### 3.4 업무공간 : 일반 지표별 표현 방식 추가 정의

#### 인당 면적

- 전체 평균값을 우선 노출
- 연도별 추세를 별도 그래프로 제공
- 값이 툴팁 없이 읽히도록 차트 내 라벨 또는 보조 표시를 적용

#### 자율좌석제 시행 비중

- 전체 도면 기준 비중은 큰 수치 카드로 제공
- 연도별 값은 별도 그래프로 제공

#### 데스크 형태별/너비별 사용 비중

- `전체 도면 기준`과 `연도별 평균 추세`를 같은 카드 안에서 함께 제공
- 차트 안에서 전체와 연도별 구간이 시각적으로 명확히 구분되어야 한다
- 값과 레이블이 가능한 한 툴팁 없이 읽히도록 직접 표기한다

#### 팀장석 구성 유형

- 전체 도면 기준은 도넛 차트
- 연도별 추세는 별도 그래프

#### 등간격 선택 순위

- 그래프 중심이 아니라 `1순위 / 2순위 / 3순위` 카드형 정보 구조를 사용
- 일자형, L자형을 분리하여 제시
- 각 순위에는 등간격 구간명과 비중을 함께 표시

#### 그룹업무공간 보유 비중

- 전체 도면 기준 값은 큰 수치 카드
- 연도별 값은 별도 그래프
- 전체와 연도별 정보가 한눈에 구분되도록 배치

## 4. 등간격 지표 계산식 확정

### 4.1 등간격 지표 산출 원칙

다음 내용을 PRD 계산식 부록에 추가한다.

> 등간격 관련 지표는 도면 단위 대표 등간격 값(`등간격_일자형`, `등간격_L자형`)을 사용한다.  
> 각 도면의 대표 등간격 값을 구간화한 뒤, 구간별 도면 수 비중으로 1순위, 2순위, 3순위를 산출한다.

### 4.2 등간격 구간 정의

최종 구간 기준:

- `1600이하`
- `1601-1750`
- `1751-1900`
- `1901-2050`
- `2051-2200`
- `2201이상`

### 4.3 등간격 지표 표시 방식

- 일자형과 L자형을 분리한다.
- 각 유형별 상위 3개 구간을 순위형으로 제시한다.
- 각 구간별 비중은 소수점 둘째 자리까지 표시한다.
