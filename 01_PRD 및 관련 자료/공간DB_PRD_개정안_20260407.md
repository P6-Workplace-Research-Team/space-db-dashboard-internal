## 1. 문서 목적

본 개정안은 향후 공간DB를 인수받는 담당자가 다음 두 가지를 빠르게 이해할 수 있도록 정리한 문서이다.

- 공간DB 대시보드의 구조와 탐색 방식
- 지표별 계산식과 산출 기준

본 문서는 기존 PRD의 내용을 대체하기보다는, `대시보드 구현 기준`과 `수정 반영 사항`을 명확히 추가한 운영용 개정안이다.

## 4. 공통 계산 원칙

### 4.1 연도 추이

- 연도 추이 차트는 `2020~2025`만 표시한다.
- `2017~2019`는 원천 데이터에는 남기되, 추세 시각화에서는 제외한다.

### 4.2 비율 지표

- 비율 추이 지표는 기본적으로 `도면별 비율`을 먼저 구한 뒤 평균한다.
- 즉, `연도 합산 후 비율`이 아니라 `도면 단위 평균`이 기본 원칙이다.

### 4.3 회의 관련 지표

- `기타회의공간`은 회의 지표에서 제외한다.
- 회의실 존재 판단 기준은 `프로그램='회의' AND 오픈실='실' AND 세부유형!='기타회의공간'`

### 4.4 지원공간 평균 개소

- 해당 공간이 존재하는 도면만 평균에 포함한다.

### 4.5 숫자 표시

- 모든 사용자 노출 숫자는 소수점 둘째 자리까지 표시한다.

## 5. 탭별 지표 정의 및 계산식

## 5.1 개괄

### 5.1.1 분석 도면 수

정의:

- 현재 필터에 포함된 도면 수

계산식:

- `COUNT(filtered drawings)`

### 5.1.2 평균 가용면적

정의:

- 현재 필터에 포함된 도면의 `가용면적` 평균

계산식:

- `AVG(가용면적)`

### 5.1.3 평균 인원수

정의:

- 현재 필터에 포함된 도면의 `인원수` 평균

계산식:

- `AVG(인원수)`

### 5.1.4 평균 인당 면적

정의:

- 도면별 `가용면적 / 인원수`를 계산한 뒤 평균

계산식:

- `AVG(가용면적 / 인원수)`

### 5.1.5 인원 구간별 도면 수

정의:

- 연도 x 인원 구간 기준 도면 수

계산식:

- `COUNT(도면 WHERE 연도=y AND 인원구간=cat)`

표현 방식:

- 자연수 히트맵 표

### 5.1.6 가용 면적 구간별 도면 수

정의:

- 연도 x 가용 면적 구간 기준 도면 수

계산식:

- `COUNT(도면 WHERE 연도=y AND 가용면적구간=cat)`

표현 방식:

- 자연수 히트맵 표

### 5.1.7 인원 구간별 가용 면적 범위

정의:

- 각 인원 구간별 가용면적의 중앙값, Q1, Q3

계산식:

- `중앙값 = P50(가용면적)`
- `적정 최소값 = P25(가용면적)`
- `적정 최대값 = P75(가용면적)`

표현 방식:

- 표 + 박스플롯

### 5.1.8 가용 면적 구간별 인원 범위

정의:

- 각 가용면적 구간별 인원수의 중앙값, Q1, Q3

계산식:

- `중앙값 = P50(인원수)`
- `적정 최소값 = P25(인원수)`
- `적정 최대값 = P75(인원수)`

표현 방식:

- 표 + 박스플롯

### 5.1.9 오피스 공간 프로그램 구성 비율 추이

정의:

- 각 도면에서 `프로그램 면적 / 도면 가용면적`을 계산한 뒤 평균

계산식:

- `AVG(프로그램 면적 / 도면 가용면적) * 100`

프로그램:

- `업무`
- `임원`
- `회의`
- `공용(화면 라벨: 소셜)`
- `지원`
- `통로 및 여유면적(화면 라벨: 기타)`

표현 방식:

- 누적 가로 막대형
- `전체` 1개 행 + `2025 → 2020` 연도별 행

## 5.2 업무공간 : 일반

### 5.2.1 인당 면적

계산식:

- `AVG(가용면적 / 인원수)`

### 5.2.2 데스크 형태별 사용 비중

정의:

- 각 도면에서 데스크 형태별 사용 인원 비중을 계산한 뒤 연도별 평균

계산식:

- `AVG(데스크형태 사용인원 / 전체 데스크 사용인원) * 100`

### 5.2.3 데스크 너비별 사용 비중

정의:

- 각 도면에서 너비별 사용 인원 비중을 계산한 뒤 연도별 평균

계산식:

- `AVG(너비구간 사용인원 / 전체 데스크 사용인원) * 100`

### 5.2.4 자율좌석제 비중

계산식:

- `COUNT(자율좌석제='O') / COUNT(전체 도면) * 100`

### 5.2.5 그룹업무공간 보유 비중

계산식:

- `COUNT(그룹업무공간 보유 도면) / COUNT(전체 도면) * 100`

## 5.3 업무공간 : 임원

### 5.3.1 오픈 임원석 보유 비율

계산식:

- `COUNT(오픈형 임원석 보유 도면) / COUNT(전체 도면) * 100`

### 5.3.2 가구구성 비율

정의:

- 실형/오픈형 각각에 대해 세부유형 매핑 기준으로 집계

비고:

- 2026-04-07 수정 시 세부유형 매핑 오류를 정정함

### 5.3.3 가구구성별 평균 면적

계산식:

- `AVG(해당 세부유형 면적)`

## 5.4 회의공간

### 5.4.1 전체 회의실 평균 개소

정의:

- 회의실이 1개 이상 존재하는 도면만 평균에 포함

계산식:

- `AVG(도면별 회의실 개수)`  
- 단, 분모는 `회의실 존재 도면`

회의실 정의:

- `프로그램='회의' AND 오픈실='실' AND 세부유형!='기타회의공간'`

### 5.4.2 연도별 / 인원구간별 / 면적구간별 회의실 평균 개소

계산식:

- `AVG(도면별 회의실 개수)`  
- 단, 해당 그룹 안에서 회의실 존재 도면만 포함

### 5.4.3 회의실 규모별 평균 개소

정의:

- `4인이하실`, `5-8인실`, `9-16인실`, `17인이상실` 각각의 평균 개소

계산식:

- `AVG(도면별 해당 규모 회의실 개수)`  
- 단, 분모는 `해당 그룹 안의 회의실 존재 도면 전체`

주의:

- `해당 규모 회의실이 있는 도면만` 평균하면 안 됨

### 5.4.4 연도별 오픈 회의 비중

계산식:

- `SUM(오픈형 회의 면적) / SUM(회의 전체 면적) * 100`
- 단, `기타회의공간` 제외

## 5.5 소셜공간

### 5.5.1 소셜 공간 정의

- 원천 데이터의 프로그램 값은 `공용`
- 화면 라벨은 `소셜`

### 5.5.2 연도별 소셜공간 유형 구성 비율

정의:

- 소셜 공간 안에서 세부유형별 면적 비중

계산식:

- 도면별 `세부유형 면적 / 공용 총면적` 계산 후 연도별 평균

### 5.5.3 라운지 계획 여부

계산식:

- `COUNT(라운지 조성 도면) / COUNT(전체 도면) * 100`

### 5.5.4 탕비 평균 면적

계산식:

- `AVG(탕비 면적)`

## 5.6 지원공간

### 5.6.1 폰부스 + 선택업무공간실 조성 비율

계산식:

- `COUNT(폰부스 또는 선택업무공간실 보유 도면) / COUNT(전체 도면) * 100`

### 5.6.2 폰부스 평균 개소

계산식:

- `AVG(도면별 폰부스 개수)`  
- 단, 폰부스가 존재하는 도면만 포함

### 5.6.3 선택업무공간실 평균 개소

계산식:

- `AVG(도면별 선택업무공간실 개수)`  
- 단, 선택업무공간실이 존재하는 도면만 포함

### 5.6.4 OA 평균 개소

계산식:

- `AVG(도면별 OA 개수)`  
- 단, OA가 존재하는 도면만 포함
