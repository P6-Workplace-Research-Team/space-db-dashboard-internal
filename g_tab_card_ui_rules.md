# G탭 카드 UI 규칙 정리

## 목적
이 문서는 현재 `G`탭 카드 시스템을 아래 기준으로 정리한 문서입니다.

1. 카드 유형
2. 현재 그 유형에 연결된 지표
3. 유형별로 의도한 UI 레이아웃 규칙

특히 어떤 카드는 핵심 값이 잘 크게 보이고, 어떤 카드는 매우 작게 보이는 문제를 파악하기 위한 기준 문서입니다.

# 모든 카드 공통 레이아웃
1. 모든 카드의 UI는 통일성을 갖추어야 합니다. 
- 카드의 유형이 동일한 경우, 카드의 UI는 동일합니다. 
- 단, 화면의 배율, 표현해야하는 내용의 길이 등에 따라 폰트의 크기는 줄어들 수 있습니다.

2. 모든 카드는 다음과 같은 항목을 포함합니다.
- 카드 제목(지표 제목) (ex. 1개층의 적정 가용 면적)
- 유형 라벨(ex. 중앙값)
  - 핵심 값을 산출하게 된 구체적인 통계값을 의미하며 다음 중 카드에 따라 하나를 택합니다. 
    - 평균, 평균 %, %, 1순위, 권장값, 선택 비율(%), 면적 비중(%)
- 핵심값 (ex. 271.75평)
  - 핵심값은 카드의 유형에 따라 산출방식이 다릅니다. (핵심 값 유형 참고)
- 보조값 (EX. 최대, 최소 값)
  - 단, 보조값은 카드의 유형에 따라 존재할 수도, 존재하지 않을 수도 있습니다.
  
3. 카드 내의 시각적 위계는 다음과 같습니다.
- 순위가 높을수록, 가장 크고 두꺼우며 시각적으로 강조되어야 합니다. 
  - 1순위 : 핵심값
  - 2순위 : 카드 제목
  - 3순위 : 보조값, 핵심값의 단위 
  - 4순위 : 핵심 값 유형 
  

4. 모든 값들은 시각적으로 읽힐 수 있는 수준의 크기여야 합니다.



## 현재 카드 렌더 함수

### 1. `guideRangeMetricCard`
- 사용 지표
  - `1개층의 적정 가용 면적`
  - `인당 면적`
  - `1개층의 적정 인원 수`
- 지표 구조
  - 핵심값: 중앙값 1개
  - 보조값: 최소값, 최대값
- 기타
  - `min`은 하단 왼쪽, `max`는 하단 오른쪽에 참고값으로 배치



### 2. `guidePercentMetricCard`
- 사용 지표
  - `업무공간 평균 면적 비중`
  - `임원 공간 평균 비율`
  - `회의 공간 평균 비중`
  - `소셜 공간 평균 비중`
  - `지원 공간 평균 비율`
  - `권장 등간격 너비`
  - `권장 열 배치`
  - `대표 가구 구성`
  - `대표 가구 구성 면적`
  - `오픈 임원석 사용도`
  - `임원 공간 평균 개소`
  - `회의실 전체 평균 개소`
  - `4인 이하 회의실`
  - `5-8인 회의실`
  - `9-16인 회의실`
  - `17인 이상 회의실`
  - `라운지 평균 개소`
  - `라운지 조성 비율`
  - `대표 라운지 유형`
  - `탕비 평균 개소 / 면적`
  - `선택업무공간_실 평균 개소`
  - `1인 공간 평균 개소 / 면적`
  - `OA 평균 개소 / 면적`
- 지표 구조
  - 핵심값 1개

### 3. `guideListMetricCard`
- 사용 지표
  - `1개층 공간 프로그램별 면적 비중`
  - `대표 데스크 형태 / 너비`
  - `권장 주통로/부통로 너비`
- 지표 구조
  - 여러 행으로 구성
  - 라벨 + 값
  - 필요 시 각 행에 보조 문구 추가
- 의도한 레이아웃 규칙
  - 각 행은 같은 기준선으로 정렬
    - 왼쪽: 라벨
    - 오른쪽: 값
  - 행 간격과 행 높이는 일정해야 함
  - 텍스트와 값은 절대 카드 밖으로 넘치면 안 됨
